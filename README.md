# Lumen Data Analysis

This project explores patterns in online content removal requests, with a focus on copyright-related takedown notices.

## Background

Online platforms receive a large number of takedown requests. Understanding how these requests are structured and distributed can provide insights into content regulation and copyright enforcement.

## Research Goals

- Analyze trends in takedown requests over time  
- Identify patterns in reporting behavior  
- Explore categories of reported content  

## Methodology

This research includes:

- Data collection from publicly available sources  
- Data preprocessing and normalization  
- Exploratory analysis  

## Current Status

Ongoing (initial phase)

## Notes

Preliminary observations suggest clustering behavior in request timestamps.
